# MicroBiz Toolbox — Complete Web Suite Architecture

## 1. Core Principle (Non-Negotiable)

This is NOT:

* 8 separate tools
* or a menu of utilities

This IS:
→ a **workflow system for freelancers to go from lead → deal → payment → repeat**

Everything must connect.

---

# 2. Global Structure (Single Website)

### Main Navigation (Left Sidebar)

* Dashboard
* Leads & Clients
* Follow-Ups
* Proposals
* Invoices
* Tasks
* Income
* Snippets
* Analytics
* Settings

👉 No more than this. If you add more, it becomes noise.

---

# 3. Central Dashboard (Home Screen = Control Center)

### What it shows (action-focused, not vanity metrics):

* “Today’s priorities”

  * Follow-ups due
  * Tasks due
  * Overdue payments

* Pipeline snapshot:

  * Leads → Active → Closed

* Money snapshot:

  * This month earned
  * Pending payments

* Quick actions:

  * Add lead
  * Send proposal
  * Create invoice

👉 If dashboard doesn’t tell user what to do in 5 seconds → it’s useless.

# 12. Cross-Tool Intelligence (Your Real Moat)

This is where most builders fail.

### Data Connections

* Lead → becomes client
* Proposal → linked to client
* Invoice → linked to proposal
* Payment → updates income

### Automation Flow Example

* Add lead
  → create follow-up reminder
  → send proposal
  → auto follow-up triggered
  → convert to deal
  → generate invoice
  → track payment

👉 If this flow is broken → product fails.
